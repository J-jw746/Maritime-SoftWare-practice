Project Name: Smart Ship Route Optimizer

Project Introduction : “Smart Ship Route Optimizer”는 선박의 운항 경로를 자동으로 최적화하여 연료 소비와 운항 시간을 최소화하는 시스템입니다.
기상 데이터(풍속, 파도 높이, 해류 방향 등)와 선박의 속도 정보를 종합적으로 분석해,
현재 해상 환경에서 가장 효율적인 항로를 추천합니다.
이 시스템은 해운 물류 효율을 높이고 탄소 배출을 줄이는 친환경 항해 지원 도구로 활용될 수 있습니다.

Main Features

기상 데이터와 해류 데이터를 기반으로 한 항로 분석

최적 경로 계산 알고리즘 구현

지도 시각화를 통한 경로 비교 표시

거리, 시간, 연료 효율 지표 제공

Tech Stack

Python (데이터 분석 및 알고리즘 구현)

OpenWeatherMap API (기상 데이터 수집)

Folium or Plotly (지도 시각화)

Flask (웹 인터페이스 구축)

Expected Outcome

효율적인 항로 선택을 통한 운항비 절감

해양 물류 산업의 에너지 절약 및 탄소 저감 효과

데이터 기반 스마트 항해 의사결정 지원
